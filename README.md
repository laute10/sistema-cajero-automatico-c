# Sistema de Cajero Automático en C

Este proyecto simula el funcionamiento básico de un cajero automático utilizando lenguaje C.

## Funcionalidades

- Inicio de sesión con usuario y contraseña
- Retiro de dinero
- Depósito de dinero
- Consulta de saldo
- Almacenamiento de datos en archivos .txt

## Tecnologías utilizadas

- Lenguaje C
- Manejo de archivos

## Cómo ejecutar

1. Compilar el archivo .c
2. Ejecutar el programa en consola
3. Asegurarse de que los archivos .txt estén en la misma carpeta

## Objetivo

Aplicar lógica de programación, estructuras de control y manejo de archivos para simular un sistema real.
